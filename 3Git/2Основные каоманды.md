1) `git init` - инициализация нового репозитория
2) ` git status`  - проверка статуса 
3) ` git add <файл>`  - добавление файлов
4) ` git commit -m "Сообщение"`  - коммит (или же -am для )
5) `git log` - просмотр логов
6) `git checkout -- <файл>` - отмена изменений в текущей директории
7) `git reset HEAD <файл>` -  удаление из staging area
