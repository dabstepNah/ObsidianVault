1) Не удается запушить в remote repo:
	1. Отсутствие прав доступа:
		- `git remote set-url origin https://username:token@github.com/user/repo.git` - перезапись URL с учетными данными
		- `git remote set-url origin git@github.com:user/repo.git` - использование SSH
	2. Локальный репозиторий отстает от удаленного:
		- `git pull origin main` - предварительный pull
	3. Неправильная настройка upstream ветки:
		- `git push -u origin main` - push в main ветку
		- `git branch --set-upstream-to=origin/main main` - установка текущей ветки на ветку main
2) Отмена всего и возврат к состоянию удаленного репозитория:
	- `git fetch origin`
	- `git reset --hard origin/main`
3) `git checkout <хэш-коммита> -- <путь-к-файлу>` - восстановление удаленного коммита
4) `git remote set-url origin git@github.com:dabstepNah/ObsidianVault.git` - сменит на пуш по SSH ключу
