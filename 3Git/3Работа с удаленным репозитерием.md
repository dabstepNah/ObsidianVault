1) `git remote add origin <url>` - добавление удаленного репозитория
2) Получение изменений:
	1. `git fetch origin` - только загрузка
	2. `git pull origin <ветка>` - fetch + merge
	3. `git pull --rebase origin <ветка>` - fetch + rebase
3) `git push origin <ветка>`  - отправка изменений
4) `git push -u origin <ветка>` - установка upstream ветки
