1) **Remote** - удаленный (внешний).
2) **Pipeline** - конвейер, по которому проходит код от коммита до продакшена.
3) **CI/CD (Continuous Integration / Continuous Deployment/Delivery)** - ключевая практика: непрерывная интеграция и непрерывное развертывание/поставка.
4) **Provisioning** - подготовка и настройка инфраструктуры.
5) **Deployment** - развертывание приложения.
6) **Availability** - доступность
7) **Disaster Recovery** - восстановление после сбоев.
8) **Infrastructure as Code (IaC)** - инфраструктура как код.
9) **Template** - шаблон.
10) **State** - состояние инфраструктуры.
11) **APM (Application Performance Monitoring)** - мониторинг производительности приложений.
12) **Vulnerability** - уязвимость.
13) **Resolution** - преобразование.
14) **Firewall** - программное, аппаратное средство, контролирующее сетевой трафик между разными сегментами сети.
