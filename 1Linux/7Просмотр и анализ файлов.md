1) **`cat` (concatenate)** — вывести содержимое файла на экран
   - `cat file.txt` — показать содержимое файла
   - `cat file1.txt file2.txt` — объединить несколько файлов
2) **`less / more`** — просмотреть файл с возможностью прокрутки
   - `less file.log` — просмотр с прокруткой (более продвинутый)
   - `more file.log` — просмотр постранично
3) **`head`** — показать первые N строк файла
   - `head -n 20 file.log` — показать первые 20 строк
   - `head file.log` — показать первые 10 строк (по умолчанию)
4) **`tail`** — показать последние N строк файла
   - `tail -n 50 file.log` — показать последние 50 строк
   - `tail -f /var/log/app.log` — следить за файлом в реальном времени
5) **`grep` (Global Regular Expression Print)** — поиск текста по шаблону
   - `grep "ERROR" app.log` — найти все строки с "ERROR"
   - `grep -r "config_value" /etc/` — рекурсивный поиск в директории
   - `ps aux | grep nginx` — поиск процесса nginx
   - `grep -v "INFO" file.log` — инвертировать поиск (строки БЕЗ "INFO")
6) **`wc` (Word Count)** — подсчет строк, слов, байт
   - `wc -l file.log` — подсчитать количество строк
   - `wc -w file.txt` — подсчитать количество слов
   - `wc -c file.txt` — подсчитать количество байт

# **Перенаправление ввода/вывода**

1) **Stdin (0)** — стандартный ввод
   - `<` — взять ввод из файла

2) **Stdout (1)** — стандартный вывод
   - `>` — передать вывод в файл (перезаписать)
   - `>>` — дописать вывод в файл

3) **Stderr (2)** — вывод ошибок
   - `2>` — перенаправить ошибки в файл
   - `2>&1` — объединить stderr с stdout

4) **| (pipe)** — отправить вывод следующей команде
   - `command1 | command2` — передать вывод command1 на вход command2

5) **tee** — отправить в файл и на стандартный вывод
   - `command | tee file.txt` — сохранить в файл и показать на экране

6) **xargs** — построчно передать на ввод команде
   - `find . -name "*.txt" | xargs rm` — удалить все найденные txt-файлы
