1) **`rsync`** — продвинутая синхронизация файлов и папок
   - `rsync -avz /local/folder/ user@remote:/remote/folder/` — синхронизация с архивацией и сжатием
2) **`history`** — история введенных команд
   - `history` — показать историю команд
   - `history | grep ssh` — найти команды ssh в истории
3) **`man (Manual)`** — встроенная справка по команде
   - `man ls` — справка по команде ls
   - `man grep` — справка по команде grep
