1) **ls** 
	- `ls -l` — подробный список
	- `ls -a` — показать скрытые файлы  
	- `ls -h` — человеко-читаемые размеры
	- `ls -t` — сортировка по времени
	- `ls -r` — обратная сортировка
	- `ls -laht` — комбинация флагов
2)  **cp** 
	- `cp -r` — рекурсивно (для директорий)
	- `cp -i` — интерактивно (спрашивать подтверждение)
	- `cp -v` — verbose (показать процесс)
	- `cp -p` — сохранить атрибуты
3)  **rm** 
	- `rm -r` — рекурсивно
	- `rm -f` — принудительно
	- `rm -i` — интерактивно
	- `rm -v` — verbose
4) **mv** 
	- `mv -i` — интерактивно
	- `mv -v` — verbose
	- `mv -n` — не перезаписывать существующие
5) Мониторинг процессов
	**ps (Process Status)**
	- `ps aux` — все процессы
	- `ps -ef` — полный формат
	- `ps -u username` — процессы пользователя
	**top**
	- `top -p PID` — следить за конкретным процессом
	- `top -u user` — процессы пользователя
	**free (Память)**
	- `free -h` — человеко-читаемый формат
	- `free -m` — в мегабайтах
	- `free -s 5` — обновлять каждые 5 секунд
6)  Поиск и обработка
	**grep (Поиск)**
	- `grep -r` — рекурсивно
	- `grep -i` — игнорировать регистр
	- `grep -v` — инвертировать поиск
	- `grep -n` — показать номера строк
	- `grep -c` — подсчитать совпадения
	- `grep -E` — расширенные regexp
	**find (Поиск файлов)**
	- `find -name "*.txt"` — по имени
	- `find -type f` — только файлы
	- `find -type d` — только директории
	- `find -mtime -7` — измененные за последние 7 дней
	- `find -size +100M` — размер больше 100MB
7)  Архивация
	**tar**
	- `tar -czf archive.tar.gz /path` — создать
	- `tar -xzf archive.tar.gz` — распаковать
	- `tar -tzf archive.tar.gz` — просмотреть
8)  Сетевые команды
	**ssh (Secure Shell)**
	- `ssh -p 2222 user@host` — нестандартный порт
	- `ssh -i key.pem user@host` — использовать ключ
	- `ssh -X user@host` — X11 forwarding
	**curl**
	- `curl -I URL` — только заголовки
	- `curl -L URL` — следовать редиректам
	- `curl -o file URL` — сохранить в файл
	- `curl -v URL` — verbose
	**wget**
	- `wget -c URL` — продолжить загрузку
	- `wget -r URL` — рекурсивно
	- `wget -O file URL` — сохранить как
9)  Справка и информация
	**Помощь**
	- `ls --help` — справка по ls
	- `grep --help` — справка по grep
	**Версии**
	- `bash --version` — версия bash
	- `git --version` — версия git
