1) **`chmod` (Change Mode)** — изменить права доступа к файлу/папке
   - `chmod +x script.sh` — сделать файл исполняемым
   - `chmod 600 private_key.pem` — установить права rw------- (только чтение/запись для владельца)
   - `chmod 755 script.sh` — права rwxr-xr-x (владелец: все, остальные: чтение и выполнение)
   - Примеры: chmod 750 script    # полные права владельцу, чтение и выполнение группе, ничего остальным
	chmod 764 script    # полные права владельцу, чтение и запись группе, чтение остальным
	chmod u+w script    # дать право записи владельцу
	chmod ugo-x script  # отобрать у всех право выполнения файла

2) **`chown` (Change Owner)** — изменить владельца файла/папки
   - `chown user:group file.txt` — изменить пользователя и группу
   - `chown -R user:group /folder/` — рекурсивно изменить владельца для папки
3) **`sudo` (Super User Do)** — выполнить команду с правами root
   - `sudo command` — выполнить команду с правами суперпользователя
   - `sudo -i` — перейти в интерактивную root-сессию
4)  Таблица комбинаций прав

| Число | Разрешение             | Символы | Описание              |
| ----- | ---------------------- | ------- | --------------------- |
| 0     | Нет прав               | `---`   | Запрет любых действий |
| 1     | Execute                | `--x`   | Только выполнение     |
| 2     | Write                  | `-w-`   | Только запись         |
| 3     | Write + Execute        | `-wx`   | Запись и выполнение   |
| 4     | Read                   | `r--`   | Только чтение         |
| 5     | Read + Execute         | `r-x`   | Чтение и выполнение   |
| 6     | Read + Write           | `rw-`   | Чтение и запись       |
| 7     | Read + Write + Execute | `rwx`   | Все права             |
5) `umask` - маска создания файлов и папок 
6) `suid` - бит запуска от имени владельца (для файлов) (значение 4)
7) `sgid` - бит запуска от имени группы владельцев (для файлов и папок) (значение 2)
8) `sticky` - бит защиты содержимого (для папок) (значение 1)
