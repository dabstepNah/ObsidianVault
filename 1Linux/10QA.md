1) `sudo -i` - выдача прав суперпользователя
2) `esc : x enter` - выход с сохранением из среды Vim
3) `chmod +x filename)` - сделать файл исполняемым
4) `./filename` - запуск исполняемого файла, каталога
5) `fmt  -(arg) (fllename)` - форматирование вывода (полезно для просмотра логов)
6)  `od` - форматирование вывода 
7) `command > filename.txt 2> fileerror` - запись в файл, иначе запись ошибки файл error
8) алгоритм создания нового профиля 
	- обращение к каталогу `/etc/skel` - системному шаблону домашних каталогов
	- настройки профиля при входе в систему`.profile`
	- настройка среды shell Bash `.bashrc`
	- `.bash_logout` скрипт, выполняемый при выходе из сеанса
9)  Создание псевдонима:
	- `vi .bashrc` (.bashrc в корневой директории пользователя)
	- Пример: `alias forhowlong='uptime -p'` (создание псевдонима для команды uptime с флагом `pretty`)
10) создание функции
	- `vi .bashrc` (.bashrc в корневой директории пользователя)
	- Пример: 
		`function wtf() {`
		`echo "your name is: ";`
		`whoami;`

		`date;`
		`echo "you are here: ";`
