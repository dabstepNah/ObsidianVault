1) **`tar` (Tape ARchive)** — архивация файлов. Используется постоянно.
   - `tar -czvf archive.tar.gz /path/to/folder/` — создать сжатый архив (gzip)
   - `tar -xzvf archive.tar.gz` — распаковать архив (extract gzip)
   - `tar -cjvf archive.tar.bz2 /path/` — создать архив с bzip2 сжатием
   - `tar -xzvf archive.tar.bz2` — распаковать bzip2 архив
2) **`gzip`** — утилита для сжатия файлов
   - `gzip file.txt` — сжать файл (создаст file.txt.gz)
   - `gunzip file.txt.gz` — распаковать файл
3) **`bzip2`** — утилита для более эффективного сжатия
   - `bzip2 file.txt` — сжать файл
   - `bunzip2 file.txt.bz2` — распаковать файл
4) **`xz`** — утилита для максимального сжатия
   - `xz file.txt` — сжать файл
   - `unxz file.txt.xz` — распаковать файл
5) **`zip/unzip`** — работа с zip архивами
   - `zip archive.zip file1 file2` — создать zip архив
   - `unzip archive.zip` — распаковать zip архив
