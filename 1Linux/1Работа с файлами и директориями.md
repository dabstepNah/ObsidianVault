1) **`pwd` (Print Working Directory)** — показать текущую директорию
   - `pwd` — отобразить полный путь текущей директории
2) **`ls` (List)** — список файлов и папок
   - `ls -la` — подробный список, включая скрытые файлы (начинаются с точки)
   - `ls -lh` — список с человеко-читаемыми размерами файлов
3) **`cd` (Change Directory)** — сменить директорию
   - `cd ~` — перейти в домашнюю директорию
   - `cd ..` — перейти на уровень выше
   - `cd -` — вернуться в предыдущую директорию
4) **`mkdir` (Make Directory)** — создать директорию
   - `mkdir new_folder` — создать новую папку
   - `mkdir -p /path/to/nested/folder` — создать цепочку вложенных папок
5) **`touch`** — создать пустой файл или обновить время его модификации
   - `touch filename.txt` — создать пустой файл
6) **`cp` (Copy)** — копировать файлы и папки
   - `cp file1.txt file2.txt` — копировать файл
   - `cp -r source_dir/ target_dir/` — копировать рекурсивно (для папок)
7) **`mv` (Move)** — переместить или переименовать файл/папку
   - `mv oldname.txt newname.txt` — переименовать файл
   - `mv file.txt /target/folder/` — переместить файл
8) **`rm` (Remove)** — удалить файлы и папки
   - `rm file.txt` — удалить файл
   - `rm -r my_dir/` — удалить папку рекурсивно
   - `rm -f file` — удалить принудительно (не спрашивая). Осторожно!
9) **`find`** — мощный поиск файлов
   - `find /path -name "*.log"` — найти все .log файлы
   - `find . -type f -mtime +7` — найти файлы, измененные более 7 дней назад
   - `find /var/log -size +10M` — найти файлы размером больше 10 МБ

# Устройство корневой папки

1) **/** — корневая файловая система, самый большой раздел
2) **/boot** — загрузочный раздел
3) **/home** — домашние папки пользователей
4) **/root** — домашняя папка суперпользователя
5) **/etc** — конфигурация системы и ее компонентов
6) **/opt** — папка для ПО от третьих поставщиков
7) **/var** — часто меняющиеся данные (логи, кэши)
8) **/usr** — все установленные пакеты программ, документация, исходный код ядра
9) **/tmp** — временные файлы
10) **Swap** — раздел подкачки (никуда не монтируется)
