1) **`uname`** — информация о системе
   - `uname -a` — показать всю информацию (ядро, версия ОС и т.д.)
   - `uname -r` — показать версию ядра
2) **`systemct`l** — управление службами и юнитами systemd
   - `systemctl status nginx` — показать статус службы
   - `systemctl start nginx` — запустить службу
   - `systemctl stop nginx` — остановить службу
   - `systemctl restart nginx` — перезапустить службу
   - `systemctl enable nginx` — включить автозагрузку службы
   - `systemctl disable nginx` — отключить автозагрузку службы
3) **`journalctl`** — просмотр логов systemd
   - `journalctl -u nginx -f` — следить за логами службы nginx в реальном времени
   - `journalctl --since "2024-01-01" --until "2024-01-02"` — логи за определенный период
   - `journalctl -p err` — показать только ошибки
4) **`crontab`** — управление периодическими задачами
   - `crontab -l` — показать текущие задачи cron
   - `crontab -e` — редактировать задачи cron
   - `crontab -r` — удалить все задачи cron
