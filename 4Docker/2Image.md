Image (образ) - это неизменный шаблон, из которого создаются контейнеры 
	Это файловая система, зависимости, runtime, код приложения, инструкции запуска 
Container (контейнер) - это запущенный экземпляр image.
Аналогия: Image - класс, container - объект
-  имеет свой `PID` namespace
-  имеет свой network namespace
-  имеет свой `filesystem` (на основе image)
-  может быть запущен / остановлен / удалён
